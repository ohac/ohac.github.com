[ソフト開発] etherealでフィルタ

ethereal の display filter で例えば HTTP の PUT method だけを表示させたいときは。

data[0:3] == 50:55:54

のように指定すればいける。0:3 は0バイト目から3バイト分という意味になる。50:55:54は"PUT"を意味する。data.startsWith("PUT")のようなことはできなさそう。

[ゲーム] ドスガレオス(MH)

やっとこさオフラインのドスガレオスを倒した。 音爆弾を大量に持ち込み挑んだが、その日の1回目は失敗。リタイアして、やり直した。

リタイアって、結構便利。持ち物がクエスト実行前の状態に戻るので、いろいろ実 験してリタイアすれば持ち物を無駄に使わずに済む。

2回目。 でかい砂漠のガレオスを全て処理してからドスガレオスに挑む。 運良く麻痺してくれたときは大タル爆弾で攻撃したりもした。 そのうち、どこかに逃げていってしまった。 どうやら体力がなくなると小さい砂漠エリアに逃げて眠ってしまうようだ。

ペイントボールを当てていたので、その場所に移動して攻撃を続けたが、 このエリアのザコは掃除していなかったので結構ジャマだった。 ネコがタル爆弾を爆発させている。おそろしや。

そんな感じでようやく倒し、☆4のクエストを受けられるようになった。コンプリートへの道は長い。ちなみにオンラインでは☆1は完了しているが、☆2がまだ残っている状況。

  • 回復笛(角笛、不死虫、竜の牙、竜の爪)

アルビノエキスってどこにあるのだろう。 調べると、フルフルから剥ぎ取りできるらしい。フルフル関連の報酬にもある。

怪力の種は火山6,7、砂漠3,5,7,10、忍耐の種は火山4にあるようだ。

笛が壊れにくくなるスキルってあるのだろうか。 採取・神のきまぐれ・幸運、精霊のきまぐれとかが関係あるのかなあ。

やっとオフモードでのリオレウスを倒した。(ポイズンタバルジン使用)

使用したトラップ系アイテムは

  • 大タル爆弾6個
  • 閃光玉5個
  • 落とし穴1個

このぐらいだったかな。

今回、わかったこと。

  • 大タル爆弾は調合前の状態で持ち込んだ方がたくさん持ち込める。
  • 大タル爆弾の着火は石だけでなくペイントボールでも可能。
  • 閃光玉はけっこう使える。
  • 閃光玉も素材玉と光蟲で持ち込んだ方がよい。
  • 解毒系のアイテムも持っておきたい。
  • 回復系アイテムは十分な量を持っておきたい。
  • マタタビも持っておきたい。
  • ビッケ(飼い猫)に注意。

[linux] ttf-kochi-mincho

debian woody 環境で ttf-kochi-mincho を使った ps ファイルを gv で見たり、gsで印字したりすると「調」の字の「言」の下の「口」の部分が塗り潰されてしまう。

そこで ttf-kochi-mincho だけを sarge や sid にすればいけるかな、と安易に考えて apt-get -t sid install ttf-kochi-mincho みたいにやってから gv で ps ファイルを見るとエラーが出る。

結局、apt-get remove ttf-kochi-mincho して、もう一度 apt-get install ttf-kochi-mincho とすると復旧した。

しかし、原因がすぐにわからず、手間取ってしまった。sarge環境ではこんなことはないのかなあ。woody から sarge へアップデートしてみたいけど、ちょっと恐い。(最初からのインストールなら恐くないけど。)

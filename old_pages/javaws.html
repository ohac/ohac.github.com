Java Web Start対応ソフト一覧

Java Web Startとは?

Javaで作成されたアプリケーションの配布、インストール、実行を簡単に実現するための仕組みです。 私の作成したアプリケーションも、Java Web Start に対応していきたいと考えております。

使い方

以下から、Java Web Start をダウンロードして、手順に従って、インストールしてください。(Javaのバージョンが1.4なら不要です。)

  • http://java.sun.com/products/javawebstart/ja/download-windows_ja.html

「ファイル」-「設定」の「詳細タブ」の「リモートアプリケーションURL」に以下を入力して、了解を押してください。

  • http://ohac.pun.jp/old_pages/javaws.html

表示されたアプリケーションを選択して、起動ボタンを押せば、自動的にダウンロードし、実行します。 2回目からはダウンロードは省略されるので、すぐに実行されます。 ただし、バージョンアップされている場合は最新版のダウンロードが行われます。
